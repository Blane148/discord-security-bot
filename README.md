# Discord Security Bot

Бот для защиты от спама через интеграции ботов.

## Функции

- Отслеживает сообщения от ботов
- Удаляет спам (5+ сообщений за 10 секунд)
- Выдает таймаут пользователям
- Отправляет логи в указанный канал

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env`:
```
DISCORD_TOKEN=ваш_токен_бота
LOG_CHANNEL_ID=id_канала_для_логов
```

## Запуск

```bash
npm start
```

Бот будет работать постоянно в фоновом режиме.
